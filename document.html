Bem-vindo à referência rápida da MySQLLib. Aqui encontras os comandos essenciais para gerir as tuas bases de dados.

Comando SELECT FROM

A introdução

SELECT * FROM tb_usuarios

é utilizada para selcecionar dados de uma base de dados

Use o asterisco (*) para selecionar todas as colunas de uma tabela

Comando INSERT INTO

Utilizado para inserir novos registros numa tabela.

INSERT INTO tb_usuarios (nome, idade)
VALUES ('Jorge Gabriel', 19)
